# generator-zfis 

> [Yeoman](http://yeoman.io) generator


## Getting Started

To install yo from npm, run:

```bash
npm install -g yo
```

### Yeoman Generators


To install generator-zfis from npm, run:

```bash
npm install -g generator-zfis
```

Finally, initiate the generator:

```bash
yo zfis
```
